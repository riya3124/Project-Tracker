🚀 Project Tracking Dashboard

📌 Overview
A smart and interactive dashboard designed to track multiple projects, monitor deadlines, manage risks, and visualize progress in real time. Built to simulate how project coordinators and managers monitor execution workflows efficiently.

✨ Key Features
- Add & manage multiple projects  
- Track project owners and deadlines  
- Risk status monitoring (Low / Medium / High)  
- Progress percentage tracking  
- Search and filter projects instantly  
- Delayed project alerts  
- Clean responsive dashboard UI  

 🛠 Tech Stack
HTML | CSS | JavaScript

📊 Use Cases
- Project Coordinator Dashboard  
- PM Intern Portfolio Project  
- Team Progress Monitoring  
- Task & Deadline Tracking    

🎯 Learning Outcome
Improved understanding of dashboard design, DOM manipulation, data filtering, project lifecycle tracking, and responsive UI development.
